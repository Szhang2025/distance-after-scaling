Diagnosing Overfitting

A model scoring 99% on training data but 60% on test data is almost certainly overfitting. It memorized the training set — including its noise — instead of learning the true pattern. This is the core bias–variance tradeoff.

Regularization

Regularization adds a penalty to the loss function discouraging large coefficients. Ridge (L2) shrinks all; Lasso (L1) zeroes some out.

More Training Data

With more data, the model has less freedom to memorize individual examples — noise averages out. Often the single most effective remedy for overfitting.

Data Augmentation Strategies

Train and test error gap shrinks as data grows. A complex model eventually generalizes with enough data.
🎲

Dropout

During each training step, randomly deactivate a fraction of neurons. Dropout prevents neurons from co-adapting and forces learning of robust, redundant representations.
